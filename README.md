# Usage

Derived from reproducer of https://github.com/oracle/graal/issues/3811

```
$ make --trace xmlformatter GRAAL=/path/to/graalvm
$ ./xslt-app-1.0 test.xml
xlst = java.io.ByteArrayInputStream@3846ae69
```
